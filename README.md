# ArchInstall
archlinux安装脚本



挂载好分区以后执行



# pacman -Sy git

把脚本下载下来


# git clone https://github.com/zhaocai-ally/ArchInstall.git


进入下载目录


# cd ArchInstall/


给脚本添加执行权限


# chmod 777 root_install.sh


执行脚本命令

# sh root_install.sh


进入硬盘后再次下载脚本文件


# git clone https://github.com/zhaocai-ally/ArchInstall.git


切换到脚本目录


# cd ArchInstall/


添加脚本执行权限


# chmod 777 yinpan_install.sh


执行脚本文件

# sh yinpan_install.sh
